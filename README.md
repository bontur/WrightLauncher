## ğŸ› ï¸ Built With

WrightLauncher leverages several powerful open-source projects and APIs:

### **Data Sources & APIs**
- **[darkseal-org/lol-skins](https://github.com/darkseal-org/lol-skins)** - Comprehensive League of Legends skins repository with all official skins in custom format
- **[noxelisdev/LoL_DDragon](https://github.com/noxelisdev/LoL_DDragon)** - Complete League of Legends Data Dragon files with champion data, images, and game assets
- **[mesalytic/discord-lookup-api](https://github.com/mesalytic/discord-lookup-api)** - Discord user lookup API for fetching user profiles, avatars, and banners
- **[Community Dragon](https://www.communitydragon.org)** - Community-driven League of Legends asset repository serving millions of requests

### **Tools & Libraries**
- **[LeagueToolkit/cslol-manager](https://github.com/LeagueToolkit/cslol-manager)** - Core skin injection and WAD file management using the Fantome mod format
- **[communitydragon/cdtb](https://github.com/communitydragon/cdtb)** - CommunityDragon Toolbox for extracting and working with League of Legends game files

### **Technology Stack**
- **.NET 8.0 & WPF** - Modern Windows desktop application framework
- **SocketIO.Client** - Real-time bidirectional communication for lobby system
- **Newtonsoft.Json** - JSON serialization for API communication
- **CommunityToolkit.Mvvm** - MVVM framework for clean architecture

## ğŸš€ Getting Started

### **For Users**
1. **Download** the latest release from the [Releases page](https://github.com/bontur/WrightLauncher/releases)
2. **Run** the installer and follow the setup wizard
